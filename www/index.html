Overview

The main purpose of Testsuite project is to provide the complex automatic tests system which is the integral part of the software development and it's quality assurance. Testsuite is a command line tool written in TCL and Expect language, a UNIX automation and testing tools.

Mission

Testsuite was originally developed for testing features and fixed bugs of Grid Engine core system nevertheless it is extendable and other related projects to Grid Engine can be tested by this tool. Currently testsuite consists of a framework and the set of tests, so called checktrees, for the following projects:

  • Grid Engine (including DRMAA)
  • ARCo
  • Hedeby

Target users

Testsuite is mainly designed for developers and engineers attached to quality assurance of projects, but also enhanced end users can use this powerful tool for instance to install the system, or at least help the open source community to improve the quality of tested products.

See the Testsuite wiki for more information.
